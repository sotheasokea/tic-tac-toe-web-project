
# Tic-Tac-Toe Game

A clean, interactive, responsive web-based Tic-Tac-Toe game built with vanilla HTML, CSS, and JavaScript. The project supports both local multiplayer and a single-player mode against a bot.

## Live Demo
You can check out the live site or share your local deployment via ngrok:
* **Live Demo Link:** `https://tic-tac-toe-web-projects.vercel.app/`
---

## Features
* **2-Player Mode:** Play locally taking turns on the same screen with a friend.
* **Play with Bot Mode:** Dynamic single-player mode against a computerized opponent using randomized decision-making.
* **Game Status Updates:** Live display indicating whose turn it is, game results, or announcing the winner (`⭕ is the winner!`).
* **Instant Reset:** Seamless "Restart Game" button functionality to quickly wipe the board and start a new match.
* **Responsive Layout:** Designed to scale beautifully across mobile and desktop browsers alike.

---

## Game Rules
1. **Player 1** always controls the `⭕` (O).
2. **Player 2** (or the computer bot) controls the `❌` (X).
3. In computer mode, you are always the first player (`⭕`) while the bot reacts immediately as the second player (`❌`).
4. The first player to align three matching symbols horizontally, vertically, or diagonally wins the match!

---

## Built With
* **HTML5:** Semantic architecture for layout controls.
* **CSS3:** Custom styles, styling effects for buttons, grid alignment, and textured board elements.
* **JavaScript (ES6+):** Handle board arrays, game states, win-condition matching logic, and bot behavior automation.
